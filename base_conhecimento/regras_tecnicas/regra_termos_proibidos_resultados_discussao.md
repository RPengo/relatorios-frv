# Regra — termos proibidos em Resultados e Discussão

- Nunca mencionar teste F, p-valor ou marcador ns no texto corrido.
- Nunca discutir coeficiente de variação (CV).
- Nunca usar "testemunha" para o tratamento Controle.
- Escrever sempre "Controle" com C maiúsculo.
- Diferenças numéricas podem ser discutidas, sem transformar ausência de significância em comprovação.
- Remover frases genéricas sobre precisão, variabilidade experimental ou CV.
